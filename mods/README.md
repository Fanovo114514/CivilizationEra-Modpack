## 📦 快速安装指南

### 方法一：使用启动器一键导入（推荐）
1. 下载 `CivilizationEra-Modpack-1.0.0.zip`
2. 打开 CurseForge / Prism Launcher / HMCL
3. 选择「导入整合包」，选择下载的 zip 文件
4. 启动器会自动下载所有模组

### 方法二：手动安装
1. 安装 Forge 1.20.1-47.3.0
2. 下载上表中所有模组 jar 文件
3. 放入 `.minecraft/mods/` 文件夹
4. 把 `config/`、`kubejs/`、`defaultconfigs/` 覆盖到对应目录
5. 把 `resourcepacks/` 中的汉化包放入资源包文件夹

---

## ⚠️ 注意事项

1. **版本对应**：请确保所有模组都是 **1.20.1 Forge** 版本
2. **前置模组**：不要忘记下载上表中的前置模组
3. **模组顺序**：一般不需要手动排序，启动器会自动处理
4. **内存分配**：建议分配 **4-6GB** 内存
5. **Java版本**：需要 **Java 17**
