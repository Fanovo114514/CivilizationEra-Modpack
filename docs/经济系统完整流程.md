# 文明纪元 - 经济系统完整使用流程

## 📌 核心概念

文明纪元的经济系统围绕**六大势力 + 动态市场 + 进化碎片**三大核心展开。
玩家通过与不同势力交易、完成任务、挖矿打怪等方式积累资源和声望，
最终推动整个文明从蛮荒时代走向星际远征。

---

## 一、六大势力与声望系统

### 1.1 势力一览

| 势力 | 定位 | 擅长领域 | 对立势力 |
|------|------|----------|----------|
| 🌾 农民联盟 (Farmers Alliance) | 农业文明 | 农作物、食物、畜牧 | 工业财团 |
| ⚒️ 矿工公会 (Miners Guild) | 采矿文明 | 矿石、金属、地质勘探 | — |
| 🏭 工业财团 (Industrial Consortium) | 工业文明 | 机械、自动化、能源 | 农民联盟 |
| 📚 学院 (Academy) | 知识文明 | 附魔、药水、研究 | 地下市场 |
| 💰 地下市场 (Underground Market) | 黑市文明 | 稀有物品、违禁品、信息 | 学院 |
| 🛡️ 守卫团 (Guard Regiment) | 军事文明 | 武器、防具、战斗 | — |

### 1.2 声望等级（从低到高）

```
仇恨 (-60) → 敌对 (-40) → 冷淡 (-20) → 中立 (0)
                                             ↓
友好 (20) ← 友善 (40) ← 尊敬 (60) ← 崇拜 (80)
```

### 1.3 声望获取方式

| 方式 | 说明 | 声望变化 |
|------|------|----------|
| 完成势力任务 | 各势力的专属任务 | +5 ~ +20 |
| 与势力交易 | 购买/出售物品 | +1 ~ +5 |
| 捐献资源 | 给势力捐物资 | +2 ~ +10 |
| 攻击势力成员 | 打了势力的人 | -10 ~ -50 |
| 抢劫势力商队 | 抢了势力的货 | -30 ~ -100 |

### 1.4 对立势力机制

> ⚔️ **关键设定**：提升某势力声望会降低其对立势力的声望（衰减50%）

- 你帮农民联盟说话，工业财团就不喜欢你
- 你跟学院混，地下市场就把你拉黑
- 矿工公会和守卫团是中立的，谁都不得罪

**举例**：你给农民联盟 +10 声望，工业财团就会 -5 声望。

## 二、动态市场系统

### 2.1 市场运作原理

市场价格不是固定的！每天都会波动，受供需关系影响：

```
价格 = 基础价格 × 价格波动系数
```

- **基础价格**：物品的标准价值（存在 JSON 配置里）
- **波动系数**：0.5x ~ 2.0x 之间每天随机浮动
- **供需影响**：卖的人多了价格下跌，买的人多了价格上涨

### 2.2 供需法则

| 行为 | 对价格的影响 |
|------|-------------|
| 大量出售某物品 | 价格下跌（供过于求） |
| 大量购买某物品 | 价格上涨（供不应求） |
| 高需求状态 (>1.3x) | 商人会喊"这玩意儿现在紧俏得很！" |
| 低需求状态 (<0.7x) | 商人会说"这货烂大街了，便宜卖" |

### 2.3 赚钱技巧 💡

1. **低买高卖**：在价格低的时候囤货，价格高的时候抛售
2. **跨势力贸易**：不同势力的物价不同，倒买倒卖
3. **关注供需**：如果大家都在卖铁锭，你就别卖了，等涨价
4. **声望加成**：声望越高，交易价格越优惠

## 三、货币系统

### 3.1 货币单位

游戏内统一使用 **金币 (Coins)** 作为货币单位。

```
1 金币 = 基础货币单位
```

### 3.2 获取金币的方式

| 方式 | 说明 | 效率 |
|------|------|------|
| 出售物品给商人 | 把资源卖给势力商人 | 中 |
| 完成势力任务 | 任务奖励金币 | 高 |
| 击杀怪物掉落 | 打怪掉金币（少量） | 低 |
| 挖矿偶然发现 | 矿洞里有金币袋 | 中 |
| 贸易差价 | 低买高卖赚差价 | 高（需要本钱） |

### 3.3 金币的用途

| 用途 | 说明 |
|------|------|
| 购买物品 | 从商人那里买物资 |
| 学习技能 | 花金币学新技能 |
| 购买蓝图 | 直接买蓝图解锁物品 |
| 捐献势力 | 捐金币换声望 |
| 建造设施 | 花钱建建筑 |

## 四、完整经济循环流程

### 🔄 新手期（蛮荒→石器时代）

```
第1步：撸树、挖石头、打僵尸
     ↓
第2步：获得进化碎片，解锁石器时代
     ↓
第3步：遇到第一个村庄/势力据点
     ↓
第4步：用手里的小麦/煤炭跟商人换第一桶金
     ↓
第5步：积累到 50 金币，开启势力声望系统
```

### 🔄 成长期（青铜→铁器时代）

```
第1步：选一个你喜欢的势力开始刷声望
     ↓
第2步：做势力日常任务，赚金币+涨声望
     ↓
第3步：声望到"友好"级别，解锁更多交易物品
     ↓
第4步：开始关注市场价格，尝试倒买倒卖
     ↓
第5步：攒够钱买第一张蓝图，提前解锁铁制工具
```

### 🔄 进阶期（蒸汽→电气革命）

```
第1步：声望达到"尊敬"，解锁势力专属商品
     ↓
第2步：建立自动化生产线（配合 Create 模组）
     ↓
第3步：大规模生产→出售→影响市场价格
     ↓
第4步：操控市场低买高卖，积累巨额财富
     ↓
第5步：花钱买通所有势力，成为商界大佬
```

### 🔄 后期（信息革命→星际远征）

```
第1步：声望全势力"崇拜"，解锁所有终极商品
     ↓
第2步：建立跨维度贸易网络（主世界→下界→末地）
     ↓
第3步：投资科研，加速科技解锁
     ↓
第4步：建造星际贸易站
     ↓
第5步：成为银河首富 💰✨
```

## 六、蓝图系统（重点！）

### 6.1 什么是蓝图？

蓝图是能让你**提前解锁单个物品**的道具。

**核心规则**：
- ✅ 时代解锁 → 该时代所有物品全部解锁
- ✅ 获得蓝图 → 即使时代没到，该物品也能制作/使用

### 6.2 蓝图获取方式

| 方式 | 概率 | 说明 |
|------|------|------|
| 挖矿（矿石） | ~3% | 挖铁矿有概率掉铁器时代蓝图 |
| 打怪（僵尸/骷髅等） | ~5% | 掉武器/防具类蓝图 |
| 村民交易 | ~10% | 交易时有概率获得交易类蓝图 |
| 宝箱 | 高概率 | 地牢、废弃矿井的宝箱里 |
| 势力商人购买 | 花钱买 | 声望够了直接买蓝图 |
| Boss 掉落 | 100% | 打 Boss 必掉稀有蓝图 |

### 6.3 蓝图使用流程

```
获得蓝图（挖矿/打怪/交易）
    ↓
自动解锁（背包里检测到就解锁）
    ↓
物品出现在 JEI 里
    ↓
可以正常合成/使用
    ↓
（时代解锁后蓝图自动失效，不重复解锁）
```

### 6.4 策略建议 💡

- **速通玩家**：疯狂挖矿打怪，赌蓝图人品，提前解锁强力装备
- **稳健玩家**：老老实实攒碎片推进时代，蓝图是锦上添花
- **商人玩家**：花钱从势力买蓝图，用钱换时间
- **欧皇玩家**：第一天就挖到钻石蓝图，直接起飞 🚀

## 七、市场交易操作指南

### 7.1 找商人

每个势力据点都有专属商人，右键点击打开交易界面。

### 7.2 交易界面

```
┌─────────────────────────────────────┐
│  🌾 农民联盟 - 老王  [友好]         │
├─────────────────────────────────────┤
│  收购价（卖给商人）                  │
│  小麦     5 金币 / 个  (需求:高)    │
│  胡萝卜   8 金币 / 个  (需求:正常)  │
│  面包    15 金币 / 个  (需求:低)    │
├─────────────────────────────────────┤
│  出售价（从商人买）                  │
│  稀有种子   50 金币 / 颗            │
│  肥料      20 金币 / 袋             │
├─────────────────────────────────────┤
│  你的金币: 230 💰                    │
└─────────────────────────────────────┘
```

### 7.3 交易技巧

1. **看需求标签**：
   - 🔴 需求高 → 商人收购价高，赶紧卖
   - 🟡 需求正常 → 价格平稳
   - 🟢 需求低 → 商人收购价低，可以买进来囤着

2. **善用声望折扣**：
   - 友善：-5% 价格
   - 尊敬：-10% 价格
   - 崇拜：-20% 价格

3. **每日刷新**：
   - 游戏内每天凌晨刷新价格
   - 蹲点等低价买入、高价卖出

## 八、经济系统与时代推进的关系

```
进化碎片（打怪/挖矿/任务）
    ↓
时代解锁（核心目标完成）
    ↓
该时代所有物品全部解锁 ✓
    ↓
更多物品 → 更多交易 → 更多金币
    ↓
更高声望 → 更好商品 → 更快推进
    ↓
（正向循环，雪球越滚越大）
```

**蓝图的作用**：在时代解锁之前，通过各种途径提前获得**单个物品**的使用权，
相当于"偷跑"——比如你还在石器时代，但挖到了铁器时代的铁镐蓝图，
那你就能提前用铁镐，挖矿效率翻倍，进化碎片赚得更快，时代推进也更快！

## 九、常见问题 FAQ

**Q: 声望负数了怎么办？**
A: 做该势力的赎罪任务，或者捐大量金币/物资挽回。或者干脆彻底加入对立势力。

**Q: 市场价格太低了卖不出去怎么办？**
A: 囤着等涨价！或者卖到别的势力那里（不同势力价格不同）。

**Q: 蓝图是消耗品吗？**
A: 不是。解锁一次永久有效，时代解锁后自动并入时代解锁。

**Q: 哪个势力最赚钱？**
A: 各有千秋。工业财团后期赚钱猛，农民联盟前期稳，地下市场风险高收益也高。

**Q: 能同时刷所有势力的声望吗？**
A: 难！因为有对立势力机制。刷农民联盟会掉工业财团的声望。
   但矿工公会和守卫团是中立的，可以一起刷。
