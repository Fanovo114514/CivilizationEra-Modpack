# 文明纪元 (Civilization Era) - 更新日志

## [1.0.0] - 2026-07-16

### ✨ 新增

- **文明核心模组 (Civilization Core)**
  - 八纪元系统：蛮荒→原始→农耕→铁器→蒸汽→电气→信息→星际
  - 进化碎片掉落与收集机制
  - 纪元任务判定与纪元切换
  - 玩家数据存储（Capability 系统）
  - 基础物品：进化碎片、铜币、银币、金币、时代核心

- **势力与经济模组 (Faction & Economy)**
  - 六大势力系统（正义/中立/邪恶各两个）
  - 势力声望与对立关系
  - 动态市场经济（价格波动、供需关系）

- **科技研究模组 (Tech Research)**
  - 科技碎片系统（5种类型：机械/电气/信息/生物/星际）
  - 蓝图与研究台系统
  - 碎片化科技树

- **蛮荒开局模组 (Barbarian Start)**
  - 蛮荒时代限制（空手不能砍树、没有床、没有船）
  - 燧石生火机制（右键石头，有成功率）
  - 原始陷阱系统（尖刺陷阱、捕兽夹）
  - 原始物品：燧石打火器、原始陷阱、石棒、肉串、骨针、皮水袋

- **KubeJS 脚本**
  - 纪元配方锁系统
  - JEI 显示控制
  - 纪元推进事件处理
  - 生物掉落进化碎片

- **FTB Quests 任务链**
  - 8个章节对应8个纪元
  - 每个纪元包含采集、制作、击杀、时代核心等任务
  - 任务奖励驱动玩家推进

- **中文汉化资源包**
  - 17个核心模组的中文汉化
  - 自创模组完整汉化

- **CurseForge 整合包**
  - 标准格式，可直接导入启动器
  - 59个第三方模组 + 4个自创模组
  - 所有配置文件预置

### 📦 配置

- 新增 7 个模组的详细配置文件
  - Waystones 传送石
  - Iron Furnaces 多级熔炉
  - Pipez 高效管道
  - Storage Drawers 抽屉存储
  - Bountiful 悬赏任务
  - YUNG's API
  - KubeJS
